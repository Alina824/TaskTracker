# Планировщик задач (Task Tracker and Calendar)

Приложение для планирования расписания на C++ с использованием Qt и SQLite. Позволяет создавать и управлять задачами с привязкой к дате и времени, просматривать расписание в недельном и месячном видах, а также вести трекер привычек. Приложение поддерживает авторизацию пользователей, систему заметок и напоминания о задачах.

Для сборки используйте CMake и Qt6. Сборка через Makefile: `make build` или `make run` для сборки и запуска.

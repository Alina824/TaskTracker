# Планировщик задач

Приложение для планирования расписания на C++ с использованием Qt и SQLite. Позволяет создавать и управлять задачами с привязкой к дате и времени, просматривать расписание в недельном и месячном форматах, а также вести трекер привычек. Приложение поддерживает авторизацию пользователей, систему заметок и напоминания о задачах.
